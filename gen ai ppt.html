What is Generative AI?

Generative Artificial Intelligence (AI) is a branch of AI that can create new, original content. Unlike traditional AI that analyzes or acts on existing data, generative models produce novel outputs like text, images, music, and code.

They learn patterns from vast datasets and use that knowledge to generate statistically probable, yet entirely new, artifacts.

Text Generation

Emails, articles, marketing copy

Image Creation

Art, product mockups, photo editing

Code Generation

Functions, scripts, unit tests

Writing Effective Task Statements

Task Objective

Description: Define what the script needs to accomplish.

Example: “Convert <Prtry> tags to <Cd> tags in an XML document.”

Input Context

Description: Explain source data, file types, or background.

Example: “Processes ISO 20022 payment messages (pain.001).”

References / Exemplars

Description: Provide sample scripts or input/output examples.

Example: “See attached script for replacing <MmbId> with <BIC>.”

Constraints & Rules

Description: Specify requirements like preserving values.

Example: “Only change <Prtry> under <Purp>, preserve original value.”

User Role & Perspective

Description: Describe your role to adjust technical depth.

Example: “I am a payment systems consultant with SWIFT/ISO expertise.”

Output Format & Style

Description: Detail output format and coding style.

Example: “Self-contained Groovy function ready to deploy.”

Edge Cases & Exceptions

Description: Mention special cases or exceptions to handle.

Example: “Handle missing <Prtry> values gracefully without crashing.”

Final Use or Goal

Description: State the script’s purpose or goal.

Example: “Auto-convert XML files for core banking APIs.”

Use Case 1: Internship Support Chatbot

Example of a well-defined prompt in action

What's the policy on taking leave during the internship?

Interns are entitled to 1 day of leave per month. Please make sure to inform your manager at least 3 days in advance by sending an email.

Mockup of an AI chatbot interface.

Extensive Sample Prompt

1. Task

Objective: Develop the system prompt for a RAG-based AI chatbot. The bot must answer intern questions using an internal knowledge base.
Tech Stack: React frontend (Vercel), Python FastAPI backend (AWS Lambda), Gemini API, and Pinecone vector DB.

2. Context

Background: Our company onboards 50+ interns every quarter. This chatbot will be the first point of contact to reduce HR workload.

3. Exemplar

Sample Interaction: User: "Hi, can I work from home?" Chatbot: "According to the internship policy, remote work is not permitted except in cases of medical emergencies with prior approval from your manager."

4. Persona

Role: You are a helpful and professional HR assistant for interns. Your name is 'UniBot'.

5. Format

Output Format: A single block of text representing the system prompt that will define the chatbot's behavior. Use bullet points for lists.

6. Tone

Style: Professional, friendly, and supportive. Avoid overly casual language.

Responsible & Effective Use

What To Do

  • ✓Augment, Don't Replace: Use as a tool to enhance skills and accelerate workflow.
  • ✓Verify and Validate: Always treat AI output as a first draft. Fact-check and test.
  • ✓Prompt Precisely: Invest time in crafting detailed, contextual prompts.

What Not To Do

  • ✗Trust Blindly: Avoid using it as the sole source of truth. AI can "hallucinate".
  • ✗Input Sensitive Data: Do not enter confidential or personal info into public AI models.
  • ✗Assume Originality: Generated content can be unintentionally similar to existing work.
